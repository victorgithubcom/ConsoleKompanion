## [1.0.0] - 2025-12-02
### Added
- Initial release of **Console Kompanion**.
- Menu-driven interface with 13 options:
  - System Info
  - File Management
  - Package Management
  - Network Basics
  - Editor Launch
  - Search Files
  - Process Manager
  - Disk Cleanup Helper
  - User Management
  - System Services (OpenRC)
  - System Monitoring
  - Customization (config file support)
  - Favorites Menu
  - Update Checker
  - Log Viewer
  - Splash Screen
  - Exit
- Default configuration file (`kompanion.conf`) with editor and folder settings.
- GPLv3 license compliance with LICENSE file and notices.
